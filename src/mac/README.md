## 键盘

### 常用按键

| Mac按键 | 键盘上标识 | 缩写 | 对应Win中的 |
| ------ | -------- | ---- | --------- |
| `Command`键 | `⌘` | Cmd | `Windows` 键 |
| `Option`键 | `⌥` |      | Alt |
| `Caps Lock`键 | `⇪` <br/> 中/英 | | |
| `Shift`键 | `⇧` | | |
| `Control`键 | `⌃` | `Ctrl` | |
| `Fn`键 | `fn` | | |
| `delete`键 | `⌫` | `del` | |

### 常用快捷键

* `Command`+`Shift`+`N`：
  * `Finder`：新建空文件夹
    * ![mac_finder_new_folder](../assets/img/mac_finder_new_folder.png)
* `Command+O`：打开所选项，或打开一个对话框以选择要打开的文件
  * 举例：
    * `Finder`：（先选中某文件后）打开文件
      * = 鼠标双击某文件
* `Command`+`↓`
  * `Finder`：（先选中某文件后）打开文件
    * = 鼠标双击某文件
* `Command+ ↑`
  * （从当前目录）回到上一级目录
